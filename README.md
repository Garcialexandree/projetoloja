# 🌿 Empório Colonial

Projeto de uma página web para um empório virtual com foco em produtos coloniais como queijos, salames, vinhos e doces.

## 🖥️ Sobre o Projeto

O **Empório Colonial** é uma aplicação front-end simples desenvolvida com HTML, CSS, JavaScript e Bootstrap, que simula uma loja online com carrinho de compras e finalização via WhatsApp.

## ✨ Funcionalidades

* ✅ Listagem de produtos
* ✅ Filtro por categorias (Queijos, Salames, Vinhos, Doces)
* ✅ Adição e remoção de itens no carrinho
* ✅ Atualização dinâmica de quantidade
* ✅ Cálculo automático do total
* ✅ Modal de carrinho
* ✅ Finalização do pedido via WhatsApp
* ✅ Interface responsiva
* ✅ Efeito visual (hover, parallax e animações)
*

## 📲 Integração com WhatsApp

O botão de finalizar pedido envia automaticamente uma mensagem com os itens selecionados para o WhatsApp.

⚠️ Para alterar o número, edite esta linha no código:

```javascript
https://wa.me/SEU_NUMERO
```
